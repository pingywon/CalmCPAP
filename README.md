CalmCPAP 0.51-alpha

Interactive tool for visualizing CPAP therapy data from SD card exports.


Features
- EDF waveform viewer
- Therapy statistics
- Daily graphs
- Machine identification display
- Onboarding agreement modal in the current standalone app

Build notes
- Vite is configured for single-file output.
- The repo currently preserves both the newer standalone snapshot and the older modular source tree for reference.
